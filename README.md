# Futhark

Простая утилита, которая кодирует короткий тег из символов `[a-z0-9]` в набор IPv6 hextet-ов и умеет декодировать его обратно.

Разработанно специально для [Seidr, Yggdrasil vanity miner](https://github.com/BITIW/Seidr) 

## Что это

`ygg-futhark` берет человекочитаемый тег, например `yggdrasil`, и превращает его в компактную шестнадцатеричную форму:

```text
yggdrasil -> 7a5e:1194:5c4b
```

При необходимости результат можно сразу оформить как заготовку под IPv6-адрес:

```text
0200:7a5e:1194:5c4b:0000:0000:0000:0000
```

## Зачем

Идея утилиты в том, чтобы:

- хранить короткий человекочитаемый тег в компактной форме;
- использовать этот тег как основу для vanity-шаблонов;
- быстро переходить из строки в hextet-ы и обратно без внешних зависимостей.

Это удобно, если хочется, чтобы тег был "виден" внутри адреса или другого шестнадцатеричного представления.

## Как это работает

Алфавит фиксированный:

```text
[a-z0-9] = 36 символов
```

Индексы такие:

```text
a = 0
b = 1
...
z = 25
0 = 26
...
9 = 35
```

Базовая идея:

- каждые 3 символа упаковываются в 1 `u16` / hextet;
- формула блока: `c1 * 36^2 + c2 * 36 + c3`;
- максимум: 7 hextet-ов, то есть до 21 символа тега.

Пример:

```text
mak
m = 12
a = 0
k = 10

12 * 1296 + 0 * 36 + 10 = 15562 = 0x3cca
```

Для формата v1 зарезервирован служебный диапазон `0x0000..=0x000f`.

- `0000` используется как padding в фиксированном формате;
- короткий последний блок кодируется отдельно, чтобы декодирование было точным даже для строк, которые заканчиваются на `a`.

## Как пользоваться

Собрать и запустить:

```bash
cargo run -- encode yggdrasil
```

Результат:

```text
7a5e:1194:5c4b
```

Фиксированный формат на 7 hextet-ов:

```bash
cargo run -- encode --fixed yggdrasil
```

Результат:

```text
7a5e:1194:5c4b:0000:0000:0000:0000
```

Полная IPv6-заготовка:

```bash
cargo run -- encode --ipv6 yggdrasil
```

Результат:

```text
0200:7a5e:1194:5c4b:0000:0000:0000:0000
```

Обратное декодирование:

```bash
cargo run -- decode 7a5e:1194:5c4b
```

или:

```bash
cargo run -- decode 0200:7a5e:1194:5c4b:0000:0000:0000:0000
```

Результат:

```text
yggdrasil
```

## Ограничения

- поддерживаются только символы `[a-z0-9]`;
- ввод автоматически нормализуется в lowercase;
- длина тега ограничена 21 символом;
- сокращенная IPv6-нотация с `::` специально не поддерживается.
